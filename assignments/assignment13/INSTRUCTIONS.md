# Module Instructions (No Terminal)

1. Open this repository in **Codespaces** from the GitHub **Code** button.
2. In the file explorer, open `student_work.py` in this folder.
3. Complete the TODOs in `student_work.py`.
4. Click the **Testing** beaker icon and run `test_assignment.py`.
5. Keep editing until all tests pass.
6. Use the **Source Control** icon to review changes.
7. Enter a commit message and click **Commit**.
8. Click **Sync Changes** to push.
9. Open or update your Pull Request in GitHub.
